# Kotlin Beginner 3. Циклы, функции и базовая автоматизация

```kotlin
fun is2xx(code: Int): Boolean {
    return code in 200..299 // Выделяем правило в функцию, чтобы переиспользовать в разных тестах.
}
```

```kotlin
val codes = listOf(200, 201, 404)   // Набор кодов для проверки.
for (code in codes) {                // Идём по коллекции без ручной индексации.
    if (is2xx(code)) println("OK")  // Используем вынесенную функцию.
}
```
