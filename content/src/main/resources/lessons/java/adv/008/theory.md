# Блок unit-архитектуры и проектирования тестов

## Урок 1. Функции, ООП и композиция
- Разделение ответственности, внедрение зависимостей.
- Тестируемые сервисы без жёсткой связанности.

## Урок 2. Mockito (отдельно)
- `mock`, `spy`, `when`, `verify`.
- Изоляция внешних API/репозиториев.

## Урок 3. Lombok (отдельно)
- `@Getter/@Setter`, `@Builder`, `@Value`.
- Когда Lombok ускоряет разработку, а когда мешает.

## Урок 4. Наследование и расширяемость
- Когда inheritance полезен.
- Проблемы глубокой иерархии и альтернатива через композицию.

## Урок 5. Исключения и их тестирование
- `assertThrows`, проверка бизнес-сообщений, негативные кейсы.
