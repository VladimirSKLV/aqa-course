# Урок 9. Gradle/Maven в AQA-проекте

Gradle и Maven решают одинаковую задачу: сборка проекта и управление зависимостями.

Что важно для автотестов:
- подключить JUnit 5;
- подключить HTTP-клиент/библиотеки валидации;
- запускать тесты командой сборщика.

Пример Gradle-задачи для тестов: `./gradlew test`.
Пример Maven-команды: `mvn test`.

Главное — договориться в команде об одном подходе и придерживаться его.
