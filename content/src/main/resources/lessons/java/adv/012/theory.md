# Урок 12. Валидация ответа

После получения ответа важно проверить:
- HTTP-статус;
- обязательные заголовки;
- структуру и значения полей JSON.

Пример проверок:

```java
assertEquals(200, response.statusCode());
assertTrue(response.body().contains("\"status\":\"OK\""));
```

Для стабильности:
- проверяйте только значимые поля;
- не сравнивайте весь JSON строкой, если порядок полей не гарантирован;
- добавляйте понятные сообщения об ошибках.
