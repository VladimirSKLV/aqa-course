# Урок 13. Подготовка к API-автоматизации

Перед полноценной API-автоматизацией подготовьте основу:

1. **Описать контракт**: какие endpoint'ы, методы, статусы и схемы данных ожидаются.
2. **Выделить тестовые данные**: что создаётся перед тестом, что очищается после.
3. **Определить структуру тестов**: smoke, regression, negative.
4. **Сделать конфиг окружений**: `baseUrl`, токены, таймауты.
5. **Подготовить отчётность**: чтобы быстро понимать, где и почему упал тест.

Цель урока — собрать фундамент, чтобы следующие API-тесты были предсказуемыми и поддерживаемыми.
