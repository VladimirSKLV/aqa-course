# Блок Selenium: отдельные уроки и домашняя работа

## Урок 1. Поиск элементов (WebDriver)
- `By.id`, `By.cssSelector`, `By.xpath`, `By.linkText`.
- Приоритет: стабильные атрибуты и читаемость.

## Урок 2. CSS-локаторы
- Атрибуты, комбинирование, псевдоклассы.
- Когда CSS быстрее и проще XPath.

## Урок 3. XPath-локаторы
- Относительный XPath, оси, предикаты.
- Антипаттерны «хрупких» цепочек.

## Урок 4. Инфраструктура браузерных тестов
- DriverFactory, конфиг `baseUrl`, таймауты, скриншоты, retries.

## Урок 5. Сложные UI-элементы и действия
- Hover, drag&drop, iframe, новые вкладки, загрузка/скачивание файлов.

## Домашнее задание
1. Написать 10 Selenium-тестов для открытого тренажёра.
2. Покрыть вкладки/фреймы/upload/download.
3. Запускать тот же набор на другом стенде через замену `baseUrl`.
