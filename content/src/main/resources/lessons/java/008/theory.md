# Урок 8. Структура папок и файлов

Новый блок: **Архитектура проекта + Gradle/Maven**.

Базовая структура AQA-проекта:
- `src/test/java` — тесты;
- `src/main/java` — вспомогательный код (клиенты, модели, утилиты);
- `src/test/resources` — тестовые данные и конфиги.

Рекомендуемые пакеты:
- `tests` — сами тесты;
- `api`/`ui` — клиентские слои;
- `config` — чтение настроек;
- `model` — DTO/POJO.

Чёткая структура ускоряет командную работу и рефакторинг.
