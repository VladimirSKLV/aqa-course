# Урок 12. Валидация ответа

После запроса валидируйте:
- статус-код;
- бизнес-поля в теле;
- негативные сценарии (например, 404).

Практика урока: написать простые проверки (assert-подобные), чтобы тест падал при неверном ответе.

Мини-пример:

```java
if (response.statusCode() != 200) {
    throw new IllegalStateException("Ожидали 200");
}
```
